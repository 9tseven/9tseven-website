# Lenis Smooth Scroll Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add global smooth scrolling to the site using Lenis via a `<ReactLenis root>` client wrapper in the layout.

**Architecture:** A single `SmoothScroll` client component wraps `{children}` in `app/layout.tsx` using `<ReactLenis root>` from `lenis/react`. Lenis drives the native `window.scrollY` directly, so motion's `useScroll` animations continue to work without changes.

**Tech Stack:** Lenis 1.3.21 (`lenis/react`), Next.js 16 App Router, React 19, TypeScript

---

### Task 1: Create the SmoothScroll client component

**Files:**
- Create: `app/components/SmoothScroll.tsx`

- [ ] **Step 1: Create the file**

```tsx
// app/components/SmoothScroll.tsx
"use client";

import { ReactLenis } from "lenis/react";

export default function SmoothScroll({ children }: { children: React.ReactNode }) {
  return <ReactLenis root>{children}</ReactLenis>;
}
```

- [ ] **Step 2: Verify TypeScript is happy**

Run: `npx tsc --noEmit`

Expected: no errors. If you see `Cannot find module 'lenis/react'`, check that `lenis` is in `node_modules` — run `npm install` if needed.

- [ ] **Step 3: Commit**

```bash

```
